## How to clone project repository
1. Create folder from terminal
```
cd /your/project/path
```
2. Clone git repositoy
```
$ git clone https://github.com/syahrillramadhan/Senbercode_Tugas_5
```

## How to run project cypress
1. install depencies
```
npm install
```
2. Open cypress.io
```
npx cypress open
```
3. Running cypress
```
npx cypress run
```
